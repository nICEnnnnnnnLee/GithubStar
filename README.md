# GithubStar
## 前言  
+ 功能  
每天记录一次指定Github Repo(此处以[BilibiliDown](https://github.com/nICEnnnnnnnLee/BilibiliDown)为例)的Star信息，  
并上传到指定路径([GithubStargazers/BilibiliDown](https://github.com/nICEnnnnnnnLee/GithubStargazers/tree/master/BilibiliDown))  

+ 实现步骤  
	+ 获取Star情况  
	+ Github文件上传  
	+ Github workfow周期性调用  

## 优势
+ 充分利用已有资源，无需额外服务器  	
+ 结果可以选择隐藏，此时将上传的目的Repo设为私有即可。
